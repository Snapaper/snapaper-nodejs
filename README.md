# Snapaper Nodejs
The Node.js Back-end of Snapaper

<br/>

## Usage
```bash
git clone git@github.com:Snapaper/snapaper-nodejs.git
```

<br/>

Make sure Node.js has been installed before you continue.
<br/>
```bash
yarn install
```

<br/>

By default, the API service will be listening to port 2333.
```bash
yarn run start
```
